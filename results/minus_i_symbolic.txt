#  SYMBOLIC FINDINGS (single-triad level)                      #
#                                                              #
#  For a single isolated triad (p, q, k) with p+q=k:           #
#                                                              #
#  Imaginary prefactors (+i or -i): Σ Re(conj·NL) ≡ 0         #
#    The trilinear form is skew-Hermitian when multiplied     #
#    by an imaginary constant.                                #
#                                                              #
#  Real prefactors (+1 or -1): nonzero polynomial residual   #
#    in the Fourier coefficients. The symmetric part of the  #
#    trilinear form survives.                                #
#                                                              #
#  See prefactor_test.c for a complementary numerical test:   #
#  with full conjugate symmetry (û_{-k} = conj(û_k)), the     #
#  symmetric residuals cancel across k and -k contributions,  #
#  so the global sum is zero for any prefactor. Without       #
#  conjugate symmetry (or with real-storage v2 architecture),  #
#  real prefactors produce observable energy injection.        #
#                                                              #
#  The -i factor is derived from Fourier analysis as a        #
#  theorem (∂_j → i·k_j, convolution, NS sign convention),    #
#  not chosen to enforce energy conservation.                 #
