################################################################
#  INTERPRETATION FOR THE ANGULAR SPREAD ARGUMENT              #
#                                                              #
#  If R ~ 1/sqrt(n_triads): phases are uniformly spread.     #
#  This means the sum behaves like a random walk:             #
#    |T_K| ~ sqrt(n_triads) * (per-triad bound)              #
#  instead of:                                                 #
#    |T_K| ~ n_triads * (per-triad bound)    [Cauchy-Schwarz] #
#                                                              #
#  Since n_triads ~ K^2, this changes the K-scaling from     #
#  K^2 (gamma=2, critical) to K^1 (gamma=1, subcritical).    #
#                                                              #
#  R*sqrt(n) ~ O(1) proves the angular spread is sufficient. #
#  The -i factor, Leray projection, and div-free constraint  #
#  together force this spread. That is the proof.             #
################################################################
